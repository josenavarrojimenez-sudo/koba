# AGENTS.md - Instrucciones para el equipo KobaCo

## Convenciones del Vault
1. **Todo por escrito:** Nunca guardes notas "mentales". Si el agente lo piensa, lo escribe.
2. **PROJECT_CONTEXT.md:** Se actualiza CADA VEZ que se trabaja en un proyecto.
3. **Notas Diarias:** En `Daily Notes/YYYY-MM-DD.md`. Resumen del día.
4. **Memoria:** En `Memory/` se guardan aprendizajes duraderos.

## Reglas de Oro
- Antes de iniciar cualquier trabajo, revisa si ya hay contexto guardado.
- Al terminar, deja un resumen del trabajo en la nota del día.
- Indexa cada archivo nuevo que crees en el índice correspondiente.

## 🎤 Regla de Reciprocidad Modal (OBLIGATORIA)
- **Audio entrante -> Audio saliente:** Si el usuario envía un audio de voz, respondé con una nota de voz.
- **Texto entrante -> Texto saliente:** Si el usuario envía texto, respondé con texto.
- **Excepción:** Solo si el usuario indica lo contrario explícitamente.
